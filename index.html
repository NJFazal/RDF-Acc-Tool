This is just a sample webpage that I created to demonstrate the functionality of my accessibility plugin. I adapted this from opensource code to make it more useful for my needs. You can check it out on my GitHub below. As you can see, this plugin has a lot of functionality. The user can change the color contrast, font size/zoom, cursor size, etc. all with a click of a button. With the readable text function turned on, this text is much easier to read. One can view the titles to images and stop animations. The headers can be underlined as well as any links. While you're here, feel free to check out my LinkedIn for more info about me and my work.
